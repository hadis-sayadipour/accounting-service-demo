# Accounting Service Demo

A simplified demonstration of an enterprise accounting platform designed for financial transaction processing, reporting, and account management.

## Features

- Journal Entry Management
- Payment Documents
- Credit Transactions
- Invoice Processing
- Account Hierarchy Management
- Financial Reporting
- REST APIs
- RabbitMQ Integration
- SQL Server Support
- MongoDB Integration

## Technology Stack

- Python
- SQL Server
- MongoDB
- RabbitMQ
- REST API

## Architecture

This project demonstrates a service-oriented backend architecture with support for:

- Financial transaction processing
- Account balance calculation
- Reporting services
- Asynchronous messaging
- Database abstraction

## Disclaimer

This repository is a simplified demonstration project based on enterprise-level accounting concepts. Sensitive business logic and proprietary company information have been removed.
